# My usb hub

## I followed the online tutorial given by hack club in the docs and decided to for the first time create a something with a pcb or schematic.

#### I began by creating the schematic and reading through the whole document and considering why i needed it. my mac mini only has 2 front facing usb c ports and no usb a ports and since i also sometimes uses windows which has usb a i needed well... more ports. so followed the guide and created the project adn worked through it to create the schematic.
